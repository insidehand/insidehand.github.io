例如說常在SwiftUI List, ForEach裡面用到的Identifiable， 每次要做比較的時候都要寫一次Equable的等價關係

如果善用protocol extenion的話，就可以針對大部分的Identifiable做一個通用的擴充

所有實作了Identifiable的對象都可以直接擴充Equable，而不用再寫同樣的程式碼


extension Equable where Self: Identifiable {
  
  static func == (lhs: Environment, rhs: Environment) -> Bool {
      lhs.id == rhs.id
  }
  
}

如果有不一樣的需求，再自己實作一次介面方法就可以了。
